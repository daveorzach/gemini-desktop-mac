# Security Policy

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report security issues privately via [GitHub's private vulnerability reporting](https://github.com/daveorzach/gemini-desktop-mac/security/advisories/new) or by emailing the maintainer directly (see GitHub profile).

Please include steps to reproduce, potential impact, and any suggested fixes if you have them. You'll receive a response within a few days.

## Scope

This app is a thin WebView wrapper around `gemini.google.com`. It does not store credentials, transmit user data, or modify Gemini's web content. Security issues in the Gemini web app itself should be reported to Google.
